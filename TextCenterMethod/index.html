文字水平居中的方法如下：

方法一： 使用 margin 属性配合 text-align 属性来使得文字居中

假设外层div元素有属性 width 为 1000px

设置文字 margin 为 0, auto

方法二： 直接设置 text-align 属性为 center

直接在文字元素上设置 text-align: center;

方法三： 设置 float 和 transform 属性为 translateX(-50%)

需要在父元素上设置 float: left; 子元素也必须设置成 position: relative; 然后再右移动父元素 50% 和左移动自身的 50%;

但是多个元素就会有问题，这时候就需要设置 clear: both; 属性

或者提前写好一个 'clearFix' 类的 CSS ，使用伪元素来清除浮动

清除浮动值得注意的是，设置 clear 属性只会影响自己

方法四： absolute 和 margin-left

在子元素上设置绝对定位，使得脱离文档流，所以需要在父元素上使用 padding 解决此问题。同时这只能是一个子元素，如果同时有另一个子元素的话需要重新布局。

这是另一个字元素

方法五： CSS3 新增的 flex 属性

在父元素中直接设置 display: flex; 和 justify-content: center; 即可。

因为比较新，所以需要注意兼容性。同时需要注意多个元素的时候。

方法六： CSS3 新增的 fit-content 属性

直接设置父元素的宽度为 fit-content 即可直接居中。

多个元素也不怕。极好！

注意，此处设置了所有的宽度都为 fit-content ，所以可以看到父元素居中后，所有子元素都居中且没有多余的宽度。
